How to use this dashboard

  1. Use the Class Codes section below to create a code and assign strands
  2. Open year5-maths-assessment.html on each student's device
  3. Students enter their name and the class code, then complete the assessment
  4. Students click "Save Results & Finish"
  5. Return to this dashboard and click Refresh
